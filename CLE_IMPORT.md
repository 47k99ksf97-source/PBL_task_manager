# CLE/Blackboardの課題を取り込む

大阪大学CLEはBlackboard Learnで、ログインや多要素認証が必要です。アプリに大学IDやパスワードを保存して直接ログインさせる方式は避けます。

このアプリでは、次の安全な取り込み方法に対応しています。

## 対応形式

- CLEのカレンダー共有リンク
- カレンダー `.ics`
- CSV `.csv`
- JSON `.json`
- CLE画面からコピーした課題一覧テキスト

読み込んだ課題は「直近の目標・提出課題」に追加されます。

## 一番かんたん: カレンダー共有リンクを登録する

1. CLEのカレンダー画面を開く。
2. 「カレンダーリンクを共有」「外部カレンダー」「iCal」などのリンクをコピーする。
3. このアプリの「CLE課題インポート」に貼り付ける。
4. `更新` を押す。

一度保存すれば、次からは `更新` を押すだけで課題期限を確認できます。

注意: カレンダー共有リンクは個人用トークンを含む場合があります。SNSやチャットに貼らないでください。

## おすすめ: カレンダーから取り込む

CLE/Blackboard側にカレンダーのエクスポートや外部カレンダーリンクがある場合は、`.ics` として保存して、このアプリの「CLE課題インポート > ファイル」から読み込んでください。

`.ics` の `SUMMARY` を課題名、`DUE` / `DTEND` / `DTSTART` を期限として読み込みます。

## コピーして取り込む

1. CLEで課題一覧やカレンダーを開く。
2. 課題名と期限が見える範囲を選択してコピーする。
3. このアプリの「CLE課題インポート > 貼り付け」を押す。

例:

```text
7/15 英語レポート提出
2026/07/20 統計学 小テスト
締切 2026-07-25 情報科学 課題3
```

## 注意

- 大学ID・パスワード・MFAコードをこのアプリに入力しないでください。
- 期限の日付が行に含まれていない課題は読み込めません。
- 既に同じ課題名と期限がある場合は重複追加しません。
