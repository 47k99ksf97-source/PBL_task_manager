# iPhoneショートカットでヘルスケアJSONを作る

このアプリは、ショートカットが作ったJSONファイルを読み込めます。

## 読み込めるJSON形式

```json
{
  "date": "2026-07-02",
  "steps": 8421,
  "sleepMinutes": 420,
  "workoutMinutes": 32,
  "activeEnergy": 510
}
```

日本語キーでも読み込めます。

```json
{
  "日付": "2026-07-02",
  "歩数": 8421,
  "睡眠分": 420,
  "運動分": 32,
  "消費カロリー": 510
}
```

## ショートカットで作る流れ

1. ショートカットアプリを開く。
2. 新規ショートカットを作る。
3. 「現在の日付」を追加する。
4. 「日付をフォーマット」で `yyyy-MM-dd` にする。
5. 「ヘルスケアサンプルを検索」で、今日の歩数を取得する。
   - 種類: 歩数
   - 開始日: 今日の開始
   - 終了日: 現在の日付
   - グループ化/合計が使える場合は合計にする。
6. 同じように、必要なら睡眠・ワークアウト時間・アクティブエネルギーも取得する。
7. 「辞書」を追加して、次のキーを作る。
   - `date`
   - `steps`
   - `sleepMinutes`
   - `workoutMinutes`
   - `activeEnergy`
8. 「辞書をJSONにエンコード」を追加する。
9. 「ファイルを保存」で `goal-task-health.json` として保存する。
10. このアプリの「ヘルスケア情報 > 読み込む」から、そのJSONを選ぶ。

## 最初に許可が必要

ショートカットを初めて実行すると、ヘルスケアへのアクセス許可が出ます。歩数・睡眠・ワークアウト・アクティブエネルギーなど、使う項目だけ許可してください。

## うまくいかない時

- JSONファイルの中身が空ではないか確認する。
- 日付が `2026-07-02` のような形式になっているか確認する。
- 睡眠やワークアウトが取れない場合は、まず歩数だけのJSONで試す。
- 読み込み後、同期欄が「同期済み」になっているか確認する。
